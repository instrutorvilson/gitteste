Inicializar repositório local
> git init

Criar nova branch
> git branch -M main

Verificar status
> git status

Associar git local a um repositório remoto
> git remote add origin https://seugit/seurepositorio

Verificar qual é o repositório remoto
> git remote -v

Adicioanr arquivo para ser versionado
> git add seuarquivo

Fazer commit 
> git commit -m "mensagem explicando a alteração"

Fazer push
> git push -u origin main
